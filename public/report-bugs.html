Bugs & Suggestions

Report bugs or recommend new features to our team

In this form, you can report bugs or suggest features for the project. Alternatively, you can visit the Github repository and submit a bug report or feature suggestion there. If you'd like to further help out the project, please consider giving a small donation.

If you have technical experience, feel free to leave pull requests on the repository with your improvements. If you would like to help the project further, please contact the owner of the website.
